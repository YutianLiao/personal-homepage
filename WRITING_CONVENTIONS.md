# Writing Conventions

This file defines style constraints for all site pages.

## Symbol Rules

- Use English punctuation only: `, . : ; ? ! ( ) [ ] { } ' " - /`.
- Do not use full-width punctuation in structural text.
- Keep list markers as `-` only.
- Use `|` as the only inline separator when needed.
- Use ASCII ellipsis `...` instead of the single-character ellipsis.

## Link Rules

- Preserve original link granularity from source material.
- Only linked source text should be wrapped in markdown links.
- Never merge adjacent links into one combined link.
- Keep non-linked text as plain text.

## Markdown Rules

- Keep heading hierarchy consistent: `#` -> `##` -> `###`.
- Keep one blank line between headings and lists.
- Prefer concise one-line bullets.
