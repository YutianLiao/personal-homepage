# Decorative character stickers

Add transparent PNG exports. If a file is missing the layout hides that image (`onerror`), so nothing breaks.

| File | Used on | Placement |
|------|---------|-----------|
| `home-hero.png` | Home (hero title) | To the **right** of the hero **name** (e.g. Yutian Liao), generous horizontal gap (`custom.css`) |
| `journey-left.png` | Interest Journey — Learning Archive | Upper-left viewport |
| `journey-right.png` | Interest Journey — Learning Archive | Mid-right viewport |
| `misc-left.png` | Miscellaneous | Bottom-left viewport |
| `misc-right.png` | Miscellaneous | Bottom-right viewport |

Recommended width for `home-hero.png`: about **120-200px** (transparent background). Corner stickers: about **220-320px** each.

**Note:** Knowledge Planet (`/interest-journey/knowledge-planet`) uses a full-width 3D canvas and does not show journey decorative stickers.
